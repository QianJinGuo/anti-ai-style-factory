# Style Seed Catalog

原创设计风格种子。每个种子是一个运动/流派的 DNA——不抄 Curio，来自设计史真实脉络。

选择标准：
- 有清晰的设计宣言或原则文献（可引用，可验证）
- 与 AI 默认输出（Inter + 紫渐变 + 玻璃拟态 + 12px 圆角）有最大反差
- 可跨场景复用（dashboard、landing、article、slide 等）
- 全部来自设计史真实运动/流派，不凭空发明

## 每个种子的产出物

一个种子通过 harness pipeline 后产出：

```
styles/{seed-id}/
  DESIGN.md        — 设计令牌规格（Google DESIGN.md 格式）
  reference.html   — 参考实现（全场景 HTML 示例）
  metadata.json    — 标签、时代、地区、适用场景、评分
  constraints.md   — 该风格特有的反 AI 味约束（在 6 金规则之上）
```

## 不做的事

- 不克隆品牌（不做 "Stripe 风格"、"Apple 风格"）
- 不照搬 Curio 的分类体系（他们的 25 标签是商业选择，不是设计史必然）
- 不凭空发明风格（全部来自设计史可查的真实运动/流派）
- 不追求数量（60 个高质量 > 500 个半成品）
