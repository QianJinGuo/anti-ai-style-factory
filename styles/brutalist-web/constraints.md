# 野蛮主义 Web — Anti-AI Constraints

These constraints apply ON TOP of the 6 golden rules.

## Style-specific rules

1. **Typography** — 系统字体
2. **Color discipline** — Derived from 野蛮主义 Web palette only
3. **Spatial logic** — Follow 原始 HTML；裸露结构；反装饰

## Forbidden (beyond 6 golden rules)

- Any visual element not justified by the 野蛮主义 Web design principle
- Decorative elements that don't serve the content structure
